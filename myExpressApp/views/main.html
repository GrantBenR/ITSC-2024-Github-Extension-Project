"What is this?"

This website is the data visualization component of our GitHub Issues Timer Extension. User, Issue, and Timer data are collected and displayed though a SQLite or POSTGRES database in tandem with our Node and Express server. Graphing is handled through Chart.js, individual records are displayed using PUG, and the SQLite database is interacted with using Prisma ORM.

"Who made this?"

This project was created as part of the Summer 2024 Software Development internship opportunity at the University of Cincinnati Information Technology Solutions Center (ITSC). Authors include GrantRynders and Deshyah with additional advising by Sil6s and ygilany

"How do I use the extension and its data tracking?"

The backend is handled locally and thus you would have to run the node server yourself in a tool like VS Code. The timer component itself can very easily be used by downloading the project from our GitHub, and loading the /Popup/ folder as an unpacked extension in Chrome (Other browsers are not yet fully supported).
